# Update Release - One Click

The release description is ready! Here's the fastest way:

## Option 1: Copy-Paste (30 seconds)

1. **Open**: https://github.com/shoutgeorge1/dj-gui/releases/edit/v1.0.0
2. **Select all** in the description box (Cmd+A)
3. **Delete** it
4. **Open** `RELEASE_DESCRIPTION_v1.0.0.md` in this folder
5. **Copy all** (Cmd+A, Cmd+C)
6. **Paste** into the GitHub description box
7. **Click** "Update release"

Done! ✅

## Option 2: Use Script (If you have a token)

```bash
# Get token from: https://github.com/settings/tokens
# Check "repo" permission
GITHUB_TOKEN=your_token python3 update_release_python.py
```
